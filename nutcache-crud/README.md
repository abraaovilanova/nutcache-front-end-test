# Nutcache crud front-end
Essa pasta do repositório contem o projeto em utilizando o `create-react-app` do front-end da aplicação

## Ferramentas utilizadas
* React
* Axios
* React-testing-library
* Jets
* msw

## Como utilizar?
* Intalação de dependencias
```bash
npm install
```
* Rodando aplicação
```bash
npm start
```
